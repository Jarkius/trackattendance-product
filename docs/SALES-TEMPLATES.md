## 1. LinkedIn Cold DM (Thai) — AV Rental / Event Production

**Target**: บริษัทรับจัดงาน, AV rental, production house ที่รับงาน BITEC / IMPACT / QSnCC

```
สวัสดีครับ/ค่ะ คุณ [ชื่อ]

เห็นว่าทีมคุณรับงาน production ที่ BITEC/IMPACT บ่อย — อยากแชร์เรื่องนึงที่น่าจะช่วยลดปัญหาหน้างานได้ครับ

ปัญหาที่เจอบ่อย: WiFi ในฮอลล์ไม่เสถียร ระบบเช็คอินค้าง คิวยาว ผู้เข้าร่วมงานรอนาน ข้อมูลหาย

ผมทำระบบเช็คอินที่ทำงานได้แม้ไม่มี WiFi ครับ:
- สแกนเข้างาน < 1 วินาที ไม่ต้องพึ่งเน็ต
- ตั้งหลายจุดสแกนพร้อมกัน sync อัตโนมัติ
- ตรวจจับ badge ซ้ำข้ามจุดได้
- Dashboard เรียลไทม์ดูจากมือถือ
- ใช้เครื่องสแกน USB ตัวละ 500 บาท

ราคาขายส่งสำหรับพาร์ทเนอร์ที่รับงานประจำ: 2,000-2,500 บาท/จุดสแกน/เดือน
(ซื้อหลายจุดลดเพิ่ม — คุยกันได้ครับ)

ขอ 3 นาทีโชว์ระบบได้ไหมครับ? ทำ demo ผ่าน Line video call ได้เลย
หรือถ้ามีงานเร็วๆ นี้ ผมไปเซ็ตให้ลองใช้ฟรีหนึ่งงานครับ

ขอบคุณครับ
[ชื่อ]
TrackAttendance
```

**Follow-up ถ้าไม่ตอบ (3 วัน)**:

```
สวัสดีครับ ส่งข้อความมาอีกทีครับ — เข้าใจว่ายุ่งช่วงเตรียมงาน

ถ้าสนใจลองดูระบบเช็คอิน offline ที่ไม่ต้องพึ่ง WiFi ฮอลล์
ผมเตรียม demo สั้นๆ 3 นาทีไว้ให้ครับ แจ้งเวลาที่สะดวกได้เลย

ขอบคุณครับ
```

## 2. LinkedIn Cold DM (Thai) — Corporate HR / Admin

**Target**: HR Manager, Admin Manager บริษัทขนาดกลาง-ใหญ่ (100-1,000+ พนักงาน)

```
สวัสดีครับ/ค่ะ คุณ [ชื่อ]

เห็นว่าคุณดูแลฝ่าย HR ที่ [ชื่อบริษัท] — อยากแนะนำระบบที่ช่วยเรื่อง attendance สำหรับงาน Town Hall, สัมมนา, หรือ training ครับ

ปัญหาที่ HR เจอบ่อย:
- จัด Town Hall ทีหนึ่ง ต้องมานั่งนับคน เช็คชื่อจาก Excel
- Training ต้องพิมพ์ sign-in sheet แล้วมา key ข้อมูลเข้าระบบ
- ไม่มีข้อมูลเรียลไทม์ว่าใครมา ใครไม่มา

TrackAttendance แก้ปัญหานี้:
- พนักงานสแกนบัตรเข้างาน — ได้ข้อมูลทันที
- ตั้งจุดสแกนกี่จุดก็ได้ sync อัตโนมัติ
- ดู dashboard จากมือถือแบบ real-time
- export Excel ได้ทันทีหลังจบงาน
- ชื่อพนักงานเก็บไว้ในเครื่องเท่านั้น ไม่ขึ้น cloud — ผ่าน PDPA

ราคา 1,500 บาท/เดือน/จุดสแกน (ยิ่งใช้หลายจุด ยิ่งถูกลง)
เทียบกับระบบ biometric ที่ต้องซื้อเครื่องตัวละ 7,000-15,000 บาท แล้วยังต้องจ่ายรายเดือนอีก

สนใจดู demo สั้นๆ ไหมครับ? ใช้เวลาแค่ 3 นาที
หรือจะลองใช้ฟรี 1 งานก่อนก็ได้ครับ

ขอบคุณครับ
[ชื่อ]
TrackAttendance
```

## 3. LINE Message (Thai) — Follow-up After Demo

**Context**: ลูกค้าดู demo แล้ว ยังไม่ตัดสินใจ — ส่ง follow-up ใน LINE

```
สวัสดีครับ คุณ [ชื่อ]

ขอบคุณที่สละเวลาดู demo ระบบเช็คอินเมื่อ [วัน] ครับ

อยากเสนอแบบนี้ครับ:

งานแรกใช้ฟรีเลย ไม่มีค่าใช้จ่าย
- ผมไปเซ็ตระบบให้หน้างาน
- ตั้งจุดสแกนเข้างานให้พร้อมใช้
- ทำ dashboard เรียลไทม์ให้ดูจากมือถือ

แลกกับขอถ่ายรูปหน้างาน 2-3 รูป + feedback สั้นๆ สำหรับทำ case study ครับ
(ไม่ต้องเปิดเผยชื่อบริษัทก็ได้ — ใช้แค่ประเภทงาน + จำนวนผู้เข้าร่วมงาน)

ถ้ามีงานเร็วๆ นี้ แจ้งวันกับจำนวนจุดสแกนที่ต้องการได้เลยครับ
ผมเตรียมให้ภายใน 1 วันทำการครับ

ขอบคุณครับ
[ชื่อ]
```

**ถ้าตอบสนใจ — ส่งต่อ**:

```
ดีเลยครับ! ขอข้อมูลเพิ่มนิดนึงครับ:

1. งานวันไหน / สถานที่
2. จำนวนผู้เข้าร่วมงานประมาณเท่าไหร่
3. ต้องการกี่จุดสแกนเข้างาน
4. มีบัตรพนักงาน/บัตร QR อยู่แล้วไหม หรือต้องพิมพ์ให้

ผมจะเตรียม proposal กับ timeline ให้ครับ
```

## 5. Cold Email (English) — Singapore / Malaysia Market

**Target**: Event managers, HR ops, coworking space operators in SG/MY

```
Subject: Offline event check-in — works when WiFi doesn't

Hi [Name],

Quick question: has your event check-in ever gone down because the venue WiFi dropped?

I built TrackAttendance specifically to solve that. It's an offline-first
badge scanning system for event check-in, corporate attendance, and training sessions.

How it works:
  - Plug in a USB barcode scanner (SGD $8), run the app, start scanning
  - Scans are instant (<1 second) — no internet needed
  - Data syncs to cloud automatically when connectivity returns
  - Set up multiple scan gates — they sync and deduplicate across stations
  - Real-time mobile dashboard — no app install, just open a link

What makes it different:
  - Offline-first: scans save locally, sync later. Zero data loss.
  - Multi-gate sync: 5 entrance gates, 1 unified headcount. No duplicates.
  - Privacy by design: attendee names stay on-premise. Only badge IDs sync.
  - No per-attendee pricing: USD $15/station/month, unlimited scans.

It's live in production today — deployed on Google Cloud (asia-southeast1),
tested with 338 automated tests, handling real events.

Typical setup takes under 5 minutes. No IT integration needed.

Would a 3-minute video demo be useful? I can also set up a free trial
for your next event — no commitment, just see if it fits.

Best,
[Name]
```

## Template Usage Guide

| Template | When to Use | Channel | Expected Response Rate |
|----------|------------|---------|----------------------|
| **1. AV Rental DM** | Prospecting production companies | LinkedIn DM | 5-15% |
| **2. Corporate HR DM** | Prospecting HR at mid-large companies | LinkedIn DM | 3-8% |
| **3. LINE Follow-up** | After demo, no decision yet | LINE | 20-40% |
| **4. Email Quotation** | After verbal agreement, need formal doc | Email | N/A (requested) |
| **5. Cold Email (EN)** | SG/MY event managers, coworking ops | Email | 2-5% |

**Tips**:
- Personalize the first line — reference their company, a recent event, or their LinkedIn post
- Thai B2B: always offer a free first event — the case study is worth more than the revenue
- Follow up exactly once after 3 days, then stop. Don't spam.
- For AV rental partners: emphasize wholesale pricing and that they can resell/bundle with their services
- For HR: emphasize PDPA compliance and zero-hardware-cost vs biometric systems
