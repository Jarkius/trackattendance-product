Offline-First Architecture

Scanning Methods

สแกน QR และบาร์โค้ด ไวปานสายฟ้า

ต่ำกว่า 0.3 วินาทีต่อคน — รองรับทุกรูปแบบการสแกน

QR Code

สแกน QR ผ่านเครื่องอ่านบาร์โค้ด USB ได้ทันที

Barcode 1D/2D

RFID / NFC

Smart Features

ฟีเจอร์อัจฉริยะที่ทำให้งานรู้สึก Premium

ระบบเสียงทักทาย อัตโนมัติ

ลดความผิดพลาด 99%

  • เสียงทักทายอัตโนมัติเมื่อสแกนสำเร็จ
  • ตั้งค่าข้อความต้อนรับ/ทักทายได้เอง
  • รองรับไฟล์เสียง MP3 และ Local TTS ใช้งาน Offline ได้

กล้องอัจฉริยะ ตรวจจับและทักทายอัตโนมัติ

เตรียมบัตรเร็วขึ้น 40%

  • Face Detection (ตรวจจับใบหน้า) — รู้ว่ามีคนอยู่หน้าจอ ด้วย MediaPipe
  • Proximity Detection (ตรวจจับระยะใกล้) — ทักทายอัตโนมัติเมื่อเดินเข้ามา
  • เปิด-ปิดแต่ละฟีเจอร์ได้อิสระ ไม่บังคับใช้กล้อง

แดชบอร์ดมือถือ ดูยอดคนได้เรียลไทม์

ไม่จำกัดจำนวนผู้เข้าชม

  • Live headcount ดูผ่านมือถือได้ทุกที่ทุกเวลา
  • แยกยอดตามหน่วยงาน + Timeline กิจกรรม 10 นาที
  • ไม่ต้อง login — แชร์ลิงก์ให้ทีมดูได้เลย

ปลอดภัยตามมาตรฐาน PDPA

Zero PII Cloud Sync

  • ข้อมูลชื่อเก็บในเครื่องเท่านั้น — ไม่ส่งขึ้นคลาวด์
  • เฉพาะรหัสบัตร (Badge ID) ที่ sync ขึ้น Cloud
  • SHA256 Encrypted Sync + Idempotent Upload

ค้นหาชื่อ-นามสกุล ไม่ใช่แค่สแกนบัตร

เช็คอินได้แม้ลืมบัตร

  • ค้นหาผู้เข้าร่วมจากชื่อ-นามสกุล แล้วเช็คอินได้ทันที
  • Fallback สำหรับผู้เข้าร่วมงานที่ลืมบัตร บัตรเสีย หรือมาแบบ walk-in
  • ค้นหาแบบ Fuzzy — พิมพ์บางส่วนก็เจอ ทำงาน Offline
