Features

ครบทุกอย่างที่ต้องการ

สแกนได้ทุกรูปแบบ

รองรับ QR Code, Barcode 1D/2D, RFID/NFC และพิมพ์รหัสด้วยคีย์บอร์ด — สแกนไม่ถึง 0.3 วินาทีต่อคน

ระบบเสียงทักทายอัตโนมัติ

เสียงต้อนรับอัตโนมัติเมื่อสแกนสำเร็จ ตั้งค่าข้อความได้เอง รองรับ MP3 และ Local TTS แบบ Offline

กล้องอัจฉริยะตรวจจับอัตโนมัติ

Face Detection + Proximity Detection ด้วย MediaPipe — ทักทายอัตโนมัติเมื่อเดินเข้ามา เตรียมบัตรเร็วขึ้น 40%

แดชบอร์ดเรียลไทม์

ดู Live headcount ผ่านมือถือได้ทุกที่ แยกยอดตามหน่วยงาน ไม่ต้อง login แชร์ลิงก์ให้ทีมดูได้เลย

ค้นหาชื่อ-นามสกุล

เช็คอินได้แม้ลืมบัตร ค้นหาแบบ Fuzzy พิมพ์บางส่วนก็เจอ รองรับ walk-in ทำงาน Offline

ปลอดภัยตามมาตรฐาน PDPA

ข้อมูลชื่อเก็บในเครื่องเท่านั้น เฉพาะ Badge ID ที่ sync ขึ้น Cloud เข้ารหัส SHA-256 ทุกครั้ง

FAQ

คำถามที่พบบ่อย

ระบบทำงานแบบ Offline ได้อย่างไร?

ข้อมูลการสแกนจะบันทึกลงเครื่องทันทีผ่าน SQLite เมื่อมีอินเทอร์เน็ต ระบบจะซิงก์ขึ้นคลาวด์อัตโนมัติ ไม่มีข้อมูลสูญหายแม้เน็ตจะล่มทั้งงาน

ต้องใช้อุปกรณ์สแกนแบบไหน?

รองรับเครื่องยิงบาร์โค้ด USB ทุกรุ่น (แนะนำ 2D scanner), กล้อง webcam สำหรับ QR Code, หรือพิมพ์รหัสด้วยคีย์บอร์ดก็ได้

ข้อมูลส่วนบุคคลปลอดภัยแค่ไหน? (PDPA)

ชื่อพนักงานเก็บไว้ในเครื่องเท่านั้น (SQLite) ไม่ขึ้นคลาวด์ ส่งขึ้นเฉพาะรหัสบัตร เข้ารหัส SHA-256 ทุกครั้งที่ซิงก์ เซิร์ฟเวอร์อยู่ใน Google Cloud ประเทศไทย (asia-southeast1)

ใช้กับงานซ้ำๆ ได้ไหม?

ได้ครับ แต่ละงานจะแยก session กัน นำเข้ารายชื่อใหม่ได้ทุกครั้ง สะดวกสำหรับงานอบรมรายเดือน หรืองาน Town Hall ประจำ

รองรับกี่สถานีพร้อมกัน?

แพ็กเกจ Starter รองรับ 3 สถานี, Professional 5 สถานี, Enterprise 10+ สถานี เพิ่มสถานีได้ไม่จำกัดในราคา 1,500 บาท/สถานี

รองรับไฟล์รูปแบบไหนสำหรับนำเข้ารายชื่อ?

รองรับ Excel (.xlsx, .xls) จับคู่คอลัมน์อัตโนมัติ รองรับหลายหน่วยงาน/แผนกในไฟล์เดียว

มีบริการติดตั้งหน้างานไหม?

แพ็กเกจ Enterprise รวมบริการติดตั้งหน้างานและ custom branding ส่วนแพ็กเกจอื่นมีคู่มือและ LINE support ตลอดงาน

ทดลองใช้ฟรีได้ไหม?

ได้ครับ แอดไลน์เพื่อนัดเดโม่ฟรี เราจะ setup ระบบทดลองให้ท่านใช้งานจริงก่อนตัดสินใจ
